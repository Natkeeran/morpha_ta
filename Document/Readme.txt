Atcharam(Tamil Morphological Analyser) 1.0

ABOUT
	Atcharam is the Morphological Analyser for Tamil. It takes an inflected word as input and separates  its root word and associated morphemes. The function of each suffix is indicated . A rule based approach is used.

REQUIREMENTS
	* J2RE 1.3
	* Operating System: Windows/Linux

Main class : Analyser.java
